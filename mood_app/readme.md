### MoodUP - Трекер настроения и эмоций
О проекте
MoodUP - это веб-приложение для отслеживания эмоционального состояния, анализа настроения и ведения дневника чувств. Приложение помогает лучше понимать свои эмоции, выявлять закономерности и поддерживать ментальное здоровье.

### Основные функции
Быстрая оценка настроения
4 типа настроения: Радостный, Спокойный, рустный, Злой

Интуитивный интерфейс с анимированными кнопками

Мгновенное сохранение настроения

Статистика и аналитика
Круговая диаграмма распределения настроений

Линейный график динамики по дням

Фильтры по периодам (неделя, месяц, год)

Аналитика и персонализированные рекомендации

Дневник настроения
Свободная форма для описания мыслей и чувств

Привязка записей к конкретным датам

Счетчик символов и автосохранение

Умные уведомления
Настройка времени получения мотивационных сообщений

Выбор тематики уведомлений (мотивация, позитив, осознанность)

Регулировка частоты отправки

Календарь настроения
Визуализация истории настроений на графике

Анализ трендов и закономерностей

Интерактивная навигация по датам

Дизайн и интерфейс
Цветовая схема: Черный, фиолетовый, белый

Адаптивный дизайн: Оптимизация для всех устройств

Анимации: Плавные переходы и интерактивные элементы

Градиенты: Фиолетовые градиенты и эффекты свечения

Технологии
Backend: Python Flask

Frontend: HTML5, CSS3, JavaScript

Визуализация: Chart.js, Plotly

Иконки: Font Awesome

Шрифты: Google Fonts (Inter)
